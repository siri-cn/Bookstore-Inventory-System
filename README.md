# Bookstore Inventory Management System

## Description
A client-server application for managing bookstore inventory using Java Swing, Socket Programming, and JDBC.

## Features
- Add new books
- View all books
- Delete books
- Real-time database updates

## Technologies
- Java
- Java Swing (GUI)
- Socket Programming
- JDBC
- MySQL Database

## How to Run
1. Create database in MySQL
2. Run Server.java first
3. Run GUI.java

## Requirements
- JDK 8+
- MySQL 8.0+
- Eclipse IDE
